HTML
HyperText MArkup LanguageHYper adalah bahasa markup yang digunakan untuk membuat dan menyusun struktur halaman WEB. HTML menggunakan elemen-elemen seperti heading, paragraf, gambar, tautan, tabel, dan daftar untuk menampilkan konten di browser.
CSS
Cascading Style Sheets adalah bahasa script yang digunakan untuk mengatur tampilan dan tata letak elemen dalam halaman web yang dibuat dengan HTML.
JavaScript
JavaScript (JS) adalah bahasa pemrograman yang digunakan untuk halaman web lebih interaktif dan dinamis. Dengan JS, kita bisa menambahkan animasi, validasi formulir, menangani event (Klik, Hover), dan banyak lagi.
